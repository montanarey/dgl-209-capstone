Design System

This design system contains code snippets and assets to be used when developing online products for Plant Parents.

By using the elements contained in this system, developers can ensure that every Plant Parents digital product is consistent in branding and style, and that each page echos the core values of Plant Parents: to make houseplant ownership easier for all.
